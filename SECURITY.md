# Security Policy

## Supported versions

Only the latest published minor release receives security fixes.

## Reporting a vulnerability

Report vulnerabilities privately through GitHub Security Advisories:
[github.com/marcop135/vite-vanilla-sass-lint/security/advisories/new](https://github.com/marcop135/vite-vanilla-sass-lint/security/advisories/new).

Do not open a public issue for security reports.
